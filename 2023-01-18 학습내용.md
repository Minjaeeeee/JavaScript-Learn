## 🔎 1. 실습 - if문, if...else문, 조건 연산자 예제
```html,js

 // 3의 배수 프로그램을 작성하여라
<div id="contents"></div>

    <script>
        var num = prompt("숫자를 입력해주세요");
        var numArea = document.querySelector("#contents");

        if(num%3==0){
            numArea.innerHTML = "3의 배수입니다"; 
        }
        else numArea.innerHTML = "3의 배수가 아닙니다.";

        
    </script>
```
